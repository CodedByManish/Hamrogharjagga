# 🏡 HamroGharJagga - Real Estate App

A modern and dynamic **Real Estate WebApp** built with **PHP & XAMPP**, featuring seamless **eSewa payment integration**. This platform connects **buyers** and **sellers** in a clean, efficient, and user-friendly environment.

---


## ✨ Features

### 🔐 Dual Authentication System

```
  👤 BUYER ACCOUNT                   🏢 SELLER ACCOUNT
──────────────────────             ──────────────────────
• Register & Login                   • Register & Login
• Search Properties                  • Add / Edit Properties
• Contact Sellers                    • Delete Listings
• eSewa Payment                      • Dashboard Management
• Browse Listings                    • Track Inquiries
```
---

## ⚙️ Setup

1️⃣ Clone Project
```bash
git clone https://github.com/CodedByManish/Hamrogharjagga.git
```

2️⃣ Move to XAMPP ```htdocs/Hamrogharjagga```

3️⃣ Start Server ```Start Apache & MySQL in XAMPP```

4️⃣ Create/Connect DB
```bash
Create DB: real_estate → Import SQL
$conn = new mysqli("localhost", "root", "", "real_estate");
```

5️⃣ Run ```http://localhost/Hamrogharjagga ```
